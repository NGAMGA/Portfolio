Site Admin Coaching — Projet de Stage

Développé dans le cadre d'un stage académique

Présentation du projet

Développement complet d’une interface administrateur pour une application de coaching sportif. L’objectif était de permettre la gestion centralisée des remboursements, des utilisateurs et des plaintes clients. En amont, une mission d’entraînement (site de restauration avec React) a servi à maîtriser le panier dynamique et le routage.

Technologies utilisées

  • React.js avec Vite : Base de l’application.
  • React Router DOM : Gestion de la navigation fluide.
  • JavaScript & CSS : Logique métier et mise en forme personnalisée.
  • Figma : Conception des maquettes visuelles.

Programmation & Hooks

  • useState : Gestion des états locaux (ex: filtres de recherche).
  • useEffect : Simulation d'appels API avec un délai de 2 secondes pour tester l'expérience utilisateur (chargement).

Fonctionnalités principales

  • Gestion des remboursements (en attente vs traités).
  • Gestion des profils utilisateurs (Coachs / Clients).
  • Système de recherche et filtrage des plaintes.
  • Sidebar de navigation et tableaux de données dynamiques.
