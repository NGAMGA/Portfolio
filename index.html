API Poker — Détails du projet

Objectif

Concevoir une API REST permettant de gérer des parties de poker : création de joueurs, distribution des cartes et simulation des tours de jeu via le terminal et Postman.

Technologies utilisées

  • Node.js
  • Express.js
  • MongoDB
  • Postman
  • Git

Fonctionnement

  • Lancement du serveur via le terminal
  • Tests des routes API avec Postman
  • Gestion des joueurs et des cartes côté backend

Défis rencontrés

  • Implémentation des règles du poker
  • Gestion des combinaisons de cartes
  • Structuration d’une API REST propre

Résultat

API backend fonctionnelle, testable entièrement via Postman, démontrant la maîtrise du développement backend.
