Movie Explorer — Laravel

Projet Full Stack Framework : Laravel 11

Objectif du projet

Concevoir une application complète de gestion cinématographique. Le projet repose sur une architecture **MVC** permettant de lister des films, de les administrer (CRUD) et de permettre aux utilisateurs de créer des collections privées et personnalisées.

Fonctionnalités clés

L'application propose deux niveaux d'accès. Tout visiteur peut consulter le catalogue et filtrer les films par catégorie. Les utilisateurs connectés bénéficient de fonctionnalités avancées :

  • Système d'Auth : Inscription, connexion et déconnexion sécurisées.
  • Administration : Gestion complète des films et des catégories (Ajout, Modification, Suppression).
  • Collections privées : Création de listes de films propres à chaque utilisateur (non visibles par les autres).
  • Données liées : Gestion des exemplaires ou notes spécifiques au sein d'une collection.

Contraintes techniques

Respect des standards académiques B3 :

  • Architecture MVC propre
  • Seeders pour l'import de données CSV
  • Base de données relationnelle
  • Design Responsive (Bootstrap)

Stack Technique

Laravel

Moteur de template Blade & Eloquent ORM

SQLite

Gestion de données via fichiers CSV

Bootstrap 5

Interface utilisateur responsive
