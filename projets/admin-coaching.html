Site Admin Coaching — Projet de Stage

Développé dans le cadre d'un stage académique

Présentation du projet

Développement complet d’une interface administrateur pour une application de coaching sportif. L’objectif était de permettre la gestion centralisée des remboursements, des utilisateurs et des plaintes clients.

Aperçu de l'interface

Technologies utilisées

  • React.js avec Vite
  • React Router DOM
  • JavaScript & CSS
  • Figma (Maquettage)

Programmation

  • useState (États locaux)
  • useEffect (Appels API)
